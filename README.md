### Installation

Clone the repository:

```bash
git clone https://github.com/kennyestrellaworks/easybank-reactjs
```

Navigate into the project directory:

```bash
cd easybank-reactjs
```

Install the dependencies:

```bash
npm install
```

### Running the Project

To start the development server:

```bash
npm run dev
```
